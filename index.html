WAD Filesystem

The culmination of my Operating Systems class was to develop a file system which would be attached to the OS of our virtual machine we were modding. In this project, we had to take a WAD File, and organize all the data into directories which were mounted onto the OS file browser. This allowed you to navigate through the WAD data as read-only files, while using the interface of the OS.
